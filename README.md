This project is a simple benchmarking tool written in C that measures and compares the execution time of four classic sorting algorithms:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
The program generates a random array of integers, runs each sorting algorithm on a copy of the same dataset, and prints the time each algorithm takes to complete.
It is designed to be clean, readable, and beginner‑friendly.

📌 Features
- Benchmarks four sorting algorithms
- Uses the same randomly generated array for fair comparison
- Clean and structured console output
- No external libraries required
- Fully written in standard C
